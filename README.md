# CoinXPintuTC

This repository is for the CoinX (PINTU) Data Analyst take-home test.

In this Repo:
- `models/raw/` : Raw data ingested from source platform.
- `models/staging/` : Staging area to cast and clean raw string data.
- `models/warehouse/` : Differentiation area between Dimension and fact tables.
- `models/marts/` : Aggregations and metrics for stakeholders needs.
